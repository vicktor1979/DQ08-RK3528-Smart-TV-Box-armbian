# DQ08 RK3528 Smart TV Box – Custom Armbian Build / Egyedi Armbian Build<p align="center">  <img src="dq08.webp" alt="DQ08 RK3528 Smart TV Box" width="500"></p>---# 🇬🇧 English## OverviewThis repository contains my customized **Armbian Build** environment for the **DQ08 RK3528 Smart TV Box**.The goal of this project is to provide a stable **Debian Bookworm** based system with full support for the DQ08 hardware and several enhancements that are not available in the stock Armbian build.---## ✨ Features* ✅ Custom DQ08 Device Tree (DTB)* ✅ RTL8733BS Wi-Fi support* ✅ RTL8733BS Bluetooth support* ✅ Automatic Bluetooth initialization during boot* ✅ OpenVFD support* ✅ `lcd-mode` utility* ✅ Optional Home Assistant Supervised installer (`install-ha`)* ✅ Custom firmware and binaries integrated through overlay* ✅ Custom kernel configuration and patches---## 📡 RTL8733BS BluetoothThis project includes a fully working implementation for the integrated RTL8733BS Bluetooth controller.Features:* Custom `rtk_hciattach`* Automatic firmware installation* Automatic configuration installation* Systemd service creation* Automatic H5 UART initialization on `/dev/ttyS2`* Fully working BlueZ integration---## 📶 RTL8733BS Wi-FiThe required RTL8733BS kernel module is automatically installed during image creation.---## 🖥️ OpenVFDSupport for the front display is included.---## 🛠 lcd-mode UtilityThe image automatically installs:```text/usr/local/bin/lcd-mode```which can be used to configure the display mode.---## 🏠 Home Assistant InstallerThe image contains an optional Home Assistant Supervised installer.It is **not executed automatically**.To start installation:```bashinstall-ha```The installer automatically:* installs Docker* installs OS Agent* installs Home Assistant Supervised* configures AppArmor* configures NetworkManager* reboots after successful installation---## 🔧 Important Kernel Configuration```textCONFIG_SERIAL_DEV_BUS=yCONFIG_SERIAL_DEV_CTRL_TTYPORT=yCONFIG_BT_HCIUART_SERDEV=yCONFIG_BT_HCIUART_3WIRE=yCONFIG_BT_HCIUART_RTL=yCONFIG_BT_RTL=y```---## 🚀 Build```bashKERNEL_BTF=no ./compile.sh \  build \  BOARD=rk3528-tvbox \  BRANCH=legacy \  BUILD_DESKTOP=no \  BUILD_MINIMAL=no \  EXPERT=yes \  KERNEL_CONFIGURE=no \  KERNEL_GIT=shallow \  RELEASE=bookworm```---# 🇭🇺 Magyar## ÁttekintésEz a repository a **DQ08 RK3528 Smart TV Box** számára készített egyedi **Armbian Build** környezetet tartalmazza.A projekt célja egy stabil, biztosít a beépített RTL8733BS Bluetooth vezérlőhöz.Automatikusan:* telepíti a módosított `rtk_hciattach` binárist,* telepíti a firmware-eket,* telepíti a konfigurációs fájlokat,* létrehozza a
